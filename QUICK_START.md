# GLOW Quick Start Guide

## For Non-Technical Users

### What is GLOW?

GLOW is your personal AI assistant for Windows. Just tell it what you want, and it does it - like having a tech-savvy friend who never sleeps!

### Installation (2 Minutes)

1. **Download** the installer: `GLOW-Setup-v1.0.5.exe`
2. **Run** the installer (double-click)
3. **Enter** your free API key when prompted
4. **Launch** GLOW from your desktop icon

That's it! No coding required.

### Getting Your Free API Key

#### Option 1: Groq (Recommended - Fastest)
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up with Google/GitHub (30 seconds)
3. Click "API Keys" ‚Üí "Create API Key"
4. Copy the key that starts with `gsk_...`
5. Paste it during GLOW installation

#### Option 2: Google Gemini (Best for Vision)
1. Go to [ai.google.dev](https://ai.google.dev)
2. Click "Get API Key"
3. Sign in with Google
4. Copy your key
5. Paste it during installation

### Understanding the Interface

#### The Glowing Orb
- **Blue**: Idle (ready for commands)
- **Green**: Thinking (processing your request)
- **Red**: Error (something went wrong)

#### Text Input
- Type your command in the text box
- Press Enter or click "Send"
- Watch GLOW work!

### Tips for Best Results

1. **Be Specific**: Instead of "open browser", say "open Chrome and go to YouTube"
2. **Chain Tasks**: "Search for Python tutorials, then open the first result"
3. **Use Vision**: "Look at my screen and tell me what you see"
4. **Save Work**: "Save this to a file called notes.txt on my desktop"

### Troubleshooting

#### "API Key Invalid"
- Check you copied the entire key (no spaces)
- Make sure the key is from the correct provider (Groq/Gemini/Claude)
- Generate a new key if needed

#### "GLOW Won't Start"
- Right-click GLOW icon ‚Üí Run as Administrator
- Check if antivirus is blocking it
- Restart your computer

#### "Command Not Working"
- Try rephrasing your request
- Be more specific about what you want
- Check the Examples section above

### Advanced Features

#### Vision Mode (Gemini Only)
GLOW can actually SEE your screen!

```
"Look at my screen and analyze this chart"
"Read the code on my screen and suggest improvements"
"Extract the key points from what's visible"
```

#### Result Chaining
GLOW remembers previous steps:

```
"Analyze the chart on my screen, summarize the findings,
 and save them to report.txt on my desktop"
```

This will:
1. Take a screenshot
2. Analyze the chart
3. Summarize the analysis
4. Create a file with the summary

### Settings

Access settings by clicking the gear icon:

- **Change AI Model**: Switch between Groq/Gemini/Claude
- **Enable Vision**: Turn vision mode on/off
- **Update API Key**: Change or add API keys

### Getting Help

- **Built-in Help**: Type "What can you do?" in GLOW
- **GitHub Issues**: [Report bugs](https://github.com/shivampal7405/GLOW/issues)
- **Documentation**: Check README.md in installation folder

### Uninstalling

1. Control Panel ‚Üí Programs and Features
2. Find "GLOW"
3. Click Uninstall
4. Follow prompts

Your config file will be removed but API keys are never shared.
